# vue-editor-lesson2-study
开发低代码平台，以数据驱动编辑器页面
低代码开发平台（LCDP）：开发者不需要传统的手写代码的方式进行编程，采用图形化拖拽的方式，配置参数完成开发工作低代码核心：降低重复劳动（营销活动、中后台系统）
组成：组件区、编辑区／预览区、属性区／事件区。通过拖拽组件生成JSON，通过JSON渲染出页面（采用JsonSchema描述页面）
